Calculate statistics for convolution layer

Example
let input = [5, 5, 3];
let filter = [2, 2, 8];
let strides = [1, 1];
let padding = [0, 0];
let stats = analyzer.analyzeConv(input, filter, strides, padding);
Parameters:
